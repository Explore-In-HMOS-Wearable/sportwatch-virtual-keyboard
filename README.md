# Virtual Keyboard

Sport watch devices do not currently support keyboards. With this demo, you can integrate a virtual keyboard into your application to get input from users.

# Use Cases

- Letter keys
- Number keys
- Symbol keys
- Space and Backspace keys

# Tech Stack

- **Languages**: ArkTS / HML
- **Frameworks**: HarmonyOS SDK 5.0.0(12)
- **Tools**: DevEco Studio 5
